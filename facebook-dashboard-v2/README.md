# Facebook Engagement Dashboard

Dashboard para analise de engajamento do Facebook, construido com React e Recharts.

## Como rodar localmente

```bash
npm install
npm start
```

Acesse http://localhost:3000

## Como fazer deploy na Vercel

1. Suba o projeto para o GitHub
2. Acesse vercel.com e conecte sua conta GitHub
3. Importe o repositorio e clique em Deploy
4. Pronto! O site estara disponivel em um link publico.

## Estrutura do projeto
